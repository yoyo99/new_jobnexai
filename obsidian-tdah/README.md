# Systeme TDAH pour Obsidian

## Installation

1. Copie le dossier `obsidian-tdah` dans ton vault Obsidian
2. Ou copie les fichiers .md directement a la racine de ton vault

## Fichiers inclus

| Fichier | Description |
|---------|-------------|
| `00-Dashboard-TDAH.md` | Tableau de bord quotidien |
| `01-Routines-TDAH.md` | Routines matin/soir/travail |
| `02-Strategies-TDAH.md` | Techniques et outils |
| `03-Template-Journee.md` | Template pour journal quotidien |
| `04-Wins-TDAH.md` | Suivi des victoires |

## Utilisation quotidienne

1. **Matin** : Ouvre le Dashboard, definis ta priorite
2. **Travail** : Utilise le Pomodoro Tracker
3. **Soir** : Note ta victoire du jour

## Plugins Obsidian recommandes

- **Templater** : Pour utiliser le template journee
- **Calendar** : Pour naviguer entre les jours
- **Tasks** : Pour gerer les taches
- **Dataview** : Pour des vues dynamiques (optionnel)

## Configuration Templater

Pour utiliser `03-Template-Journee.md` avec Templater :
1. Va dans Settings > Templater
2. Definis le dossier de templates
3. Cree une nouvelle note avec le raccourci Templater
